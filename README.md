# Learn Everything — 用 AI 的方式学习。

**6 种风格的学习方式、费曼检查、自动笔记、自动记录学习进度。**

> 把文档、项目、课程扔进来。AI 老师帮你拆解、备课、讲课，然后考你，直到你真会了。

---

## 为什么你需要这个

和 AI 聊天问问题 ≠ 学习。你问一句它答一句，聊完就忘了，知识点散落在聊天记录里，下次打开从头来。

**Learn Everything 做的不是"对话"，是"教学"：**

```
你: 教我 Kubernetes
AI: （分析内容，拆成 7 节课）
    ├── 00-课程概述.md        ← 课程全景
    ├── 01-什么是K8s.md        ← 完整教材
    ├── 02-Pod详解.md
    ├── 03-Service与网络.md
    ...
    （全部教材写好后）准备好了，开始第一课吗？
```

讲完一节 → 自评"会了吗？" → 费曼检查"你用自己的话解释一下" → 小测验 → 答错了自动记到笔记 → 问"继续下一课？"

你学到的东西不是散在聊天里的，是**一本完整的教材**。`.md` 文件，是你的，打开就能读。

---

## 6 位老师，6 种学法

| 老师 | 怎么教 | 适合 |
|------|--------|------|
| **学院派教授** | 追根溯源，画架构图，讲为什么 | 系统性学习理论 |
| **实战派导师** | 代码先行，例子驱动，边做边讲 | 学技术工具/框架 |
| **苏格拉底导师** | 全程反问，不直接给答案 | 想挑战深度理解 |
| **技术文档向导** | 逐节解读官方文档 | 啃官方文档 |
| **结对编程同事** | 一起读源码、调试、改代码 | 学开源项目 |
| **TL;DR 摘要员** | 15 分钟抓 20% 核心骨架 | 快速建立全局认知 |

**教材是完整版，老师只决定讲多深。** 换老师不丢进度——先用摘要员扫一遍，再切教授深入学，同一个知识图谱。

## 你学到的东西长什么样

```
.learning/kubernetes/
├── 00-课程概述.md                    ← 课程全景 & 知识图谱
├── 01-概述：什么是Kubernetes.md       ← 第 1 节课教材（打开就能读）
├── 02-Pod详解.md                     ← 第 2 节课教材
├── 03-Service与网络.md
├── ...
├── progress.md                       ← 学习状态 & 评分
├── transcript.jsonl                  ← 完整互动记录（每条都有）
├── notes.md                          ← 自动总结 + 错题集 + 你的随手笔记
```

- **`.md` 课程文件**：你的教材。稳定的、结构化的、打开就能重读
- **`transcript.jsonl`**：每一条互动记录（提问、费曼、测验），可搜索可分析
- **`notes.md`**：每节自动总结 + 答错的题 + 你手动记的笔记

## 怎么用

### 一键安装（Claude Code）

```bash
bash install.sh
```

自动完成三件事：软链接技能 → 创建配置文件 → 注册写入强制 Hook。然后直接说"教我 React"就行。

### 手动安装（Cursor / Codex）

- **Cursor**：复制 `adapters/cursor.md` 的内容到 `.cursorrules` 或 Custom Instructions
- **Codex**：复制 `adapters/codex.md` 的内容到 Session Instructions

### 配置

学习进度文件默认存 `.learning/`。在 `config.json` 中修改 `learning_dir`。

### Hook 强制落盘（Claude Code 专属）

安装时自动注册 `UserPromptSubmit` Hook。每次上课（`phase=learning`），模型收到你的消息前都会看到这张强制提醒表：

| 你做了什么 | 必须立即 Write |
|-----------|-----------|
| 和学生互动 | transcript.jsonl |
| 回答了 !ask | transcript.jsonl + 判断是否记 note |
| 费曼 / 测验 | transcript.jsonl |
| 测验答错 | notes.md（错题集） |
| 一节结束 | notes.md 总结 + 问下一课 |

只在学习会话中生效，其他对话零开销。

## 和"直接问 ChatGPT"有什么区别

| | 问 ChatGPT | Learn Everything |
|---|-----------|-----------------|
| 内容 | 散落在聊天里 | 写成 `.md` 教材文件，你的 |
| 检验 | 你觉得懂了就是懂了 | 费曼检查 + 测验，不过关重来 |
| 进度 | 每次从头聊 | 明天打开继续学 |
| 风格 | 一种 | 6 种老师可选 |
| 笔记 | 手动复制 | 自动总结 + 错题自动记录 |
| 交互 | 一问一答 | 备课-上课-检查-测验-下一课 |
